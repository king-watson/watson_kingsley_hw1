# SWAPI AJAX REDUX

## Overview üöÄ 
This repository contains a Star Wars Character & Movie Guide, developed as part of my Web Development coursework. The goal of this project is to research and consume a public API (SWAPI), dynamically fetch data using JavaScript, and present it in a responsive, interactive interface.

The application allows users to browse a list of Star Wars characters and click on a character to view detailed movie information ‚Äî including the movie title, opening crawl, and poster ‚Äî all loaded dynamically using AJAX requests.

### Requirements Addressed ‚úÖ
API Data Consumption (SWAPI)

Characters are fetched from the Star Wars API (https://swapi.info)
A minimum of 10 characters are dynamically populated into an unordered list
Dynamic Character List

Each list item displays:

- Character name
- A clickable link tied to a movie the character appears in
- No hard-coded character data
- Second AJAX Call (Movie Data)
- When a character is clicked:
- A second fetch request is triggered

The following movie details are displayed in a separate section:
- Movie title
- Opening crawl
- Movie poster (stored locally in the /images folder)
- Graceful Fetch Handling

All AJAX requests handle different states:
- Loading state with a loading icon
- Successful data retrieval
- Error handling for failed requests
